# PTCGP Tracking Spreadsheet Importer

PTCGP Tracking Spreadsheet Importer is a Google Sheets tool that allows users to transfer checkbox and quantity data from an older tracking spreadsheet into a newer version of the same spreadsheet, due to the spreadsheet being constantly updated as its source material, Pokémon TCG Pocket, is.

## What the Application Does

- Reads data from a user-selected Google Spreadsheet
- Copies checkbox values and/or quantity values, at the user's selection
- Writes that data into the currently active spreadsheet

## Google Data Access

This application requests Google Sheets and Google Drive permissions only to read and write spreadsheet data selected by the user.

No data is stored outside of Google services.
No data is sold, shared, or transferred to third parties.
